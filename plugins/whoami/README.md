## WhoAmI: Request Header Parser Microservice

## User stories:
1. I can get the IP address, language and operating system for my browser.

## Example usage:

```
http://localhost:3000/api/whoami
```

## Example output:

```js
  {
    "ipAddress": "121.0.0.1",
    "language": "en-US",
    "software": "Macintosh; Intel Mac OS X 10_11_4"
  }
```
